# AegisScan
AegisScan是一款“自动化”的“渗透测试”扫描工具。

使用场景：

护网、攻防演练快速抢分使用

红队攻击人员日常渗透测试使用

蓝队防守人员日常巡检使用

小白、懒人使用工具：无任何参数，不需要指定繁杂参数，回车就自动进行测试。

如何配置一个钉钉群聊机器人，以便它可以发送消息到钉钉群聊？

操作步骤

1.打开钉钉客户端，进入指定群，单击群设置。
单击群管理 > 机器人 > 添加机器人。

2.选择合适的机器人进行添加：
      阿里云 Codeup 机器人：阿里云持续交付平台提供的代码托管服务。
      GitHub 机器人：基于Git的代码托管服务。
      极狐 GitLab 机器人：基于ROR的开源代码托管软件。
      JIRA 机器人：出色的项目与事务跟踪工具。
      Travis 机器人：出色的项目与事务跟踪工具。
      
3.获取机器人 Webhook 地址。

4：把上面的webhook URL和 secret的值填入到config.json的对应位置。

4、版本说明：

由于扫描是大并发，对内存和网络有一定要求，特别是网络带宽，如果网络拥塞，就会产生漏报之类的。

如果您的机器内存是16G，建议是32G，网络带宽很好的话，建议运行AegisScan16G.exe这个版本，这个扫描起来最快，节省时间。

如果您的机器内存是8G左右，网络带宽一般的话，建议运行AegisScan8G.exe这个版本，扫描出结果的速度居中。

如果您的机器内存是8G以下，或者4G的老机器，网络带宽一般的话，建议运行AegisScan4G.exe这个版本，，扫描出结果的时间就相对慢一些，但是漏报肯定少。

如果你的机器配置很高，网络带宽也很好，但是又不急着出结果，想降低漏报，以时间换取结果，建议运行AegisScan4G.exe或8G的这个版本。
